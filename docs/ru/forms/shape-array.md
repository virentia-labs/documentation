# Динамические поля

Эта страница оставлена как карта для динамических структур формы. Паттерны
разнесены по отдельным страницам:

- [Shape-поля](./shape-fields) - динамические ключи объекта.
- [Array-поля](./array-fields) - упорядоченные повторяемые элементы.

Берите shape-поля, когда пользователь редактирует объект-словарь.
Берите array-поля, когда важны порядок и операции над элементами.
